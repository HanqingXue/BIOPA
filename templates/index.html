Pathway Visualization

The main function of this module is to display the retrieved information in a network. These sub-modules are integrated into the toolbar for use by users.

The basic functions of the module include network amplification, network reduction, network export, network layout adjustment, node display, node hiding, network analysis, network maximization

General Information Display

This panel is designed to provide users with intuitive information display

We provide four sub-modules to display and edit the informations of networks

Basic functions include node information display, side information display, network editing template, side information statistics

Detail Information Display

This module realized the visualization of biological networks

The module provides six sub-modules: gene detailed information display, disease detailed information display, drug information display, pathway information display, network analysis result display, and gene expression information display module.
